# 加班记录工具 - README

## 概述

加班记录工具是一个基于Web的应用程序，专为记录和管理加班时间、请假以及补贴扣款信息而设计。它提供了直观的日历视图、详细的统计信息和Excel导入导出功能，帮助用户轻松跟踪工作时间和计算相关薪资。

## 🌟 主要功能

### 📅 智能日历视图
- **直观月历显示**：支持年份和月份快速切换
- **智能日期标记**：不同颜色区分节假日、周末、调休日和特殊工作日
- **一键记录**：直接点击日期记录加班或请假信息
- **今日高亮**：当前日期特殊标识，便于快速定位

### 💰 精准薪资计算
- **灵活工资设置**：支持自定义标准工资（默认4600元）
- **自动薪资计算**：基于国家标准21.75个工作日计算日薪和时薪
- **智能倍率应用**：
  - 🟢 工作日加班：1.5倍
  - 🔵 周末加班：2倍  
  - 🟣 节假日加班：3倍
  - 🟡 调休日加班：2倍
  - 🔴 请假扣款：1.5倍

### 📊 全面统计功能
- **时长统计**：总加班时长和净加班时长（扣除请假）
- **请假管理**：请假时长统计和自动扣款计算
- **分类统计**：按加班类型（A/B/C/特）分别统计薪资
- **实发工资**：自动计算实发金额（标准工资 + 加班费 + 补贴 - 请假扣款）

### 💸 智能补贴扣款管理
- **多分类支持**：交通补贴、餐补、水电费、绩效、公积金、社保等
- **自定义分类**：支持添加个性化分类
- **正负记录**：补贴（正数）和扣款（负数）统一管理
- **历史追溯**：完整的记录查看和编辑功能

### 🔄 强大数据管理
- **Excel导出**：生成专业Excel报表（加班记录、补贴扣款、统计汇总）
- **数据导入**：支持从Excel文件恢复数据
- **批量操作**：一次性设置多个日期的加班时长
- **本地存储**：数据自动保存，无需手动操作

### 🔒 高级隐私保护
- **一键隐藏**：快速隐藏所有金额信息
- **选择性隐藏**：可单独隐藏特定统计项
- **触摸优化**：移动设备完美支持
- **安全可靠**：数据仅存储在本地浏览器

## 🚀 使用指南

### 基本操作
1. **设置基础工资**：在顶部输入框调整您的月薪
2. **记录考勤**：
   - 点击日历日期，输入正数记录加班
   - 输入负数记录请假  
   - 输入0清除记录
3. **管理补贴**：使用"补贴/扣款"功能记录额外收支
4. **查看统计**：实时查看薪资计算结果

### ⚡ 批量操作技巧
1. 点击"批量设置加班"进入批量模式
2. 在日历中多选日期（支持取消选择）
3. 输入统一时长，一键应用到所有选中日期
4. 支持加班和请假的批量设置

### 📤 数据备份与恢复
- **导出备份**：每月生成包含完整数据的Excel文件
- **导入恢复**：从备份文件快速恢复所有记录
- **跨月管理**：支持不同月份数据的独立管理

### 👁️ 隐私保护使用
- **全局隐藏**：点击"隐藏金额"按钮保护隐私
- **精细控制**：点击各统计项的眼睛图标单独控制
- **移动友好**：在公共场所使用更安心

## 📝 计算规则说明

### 薪资计算基础
- **月工作日**：21.75天（国家标准）
- **日薪计算**：月薪 ÷ 21.75
- **时薪计算**：日薪 ÷ 8小时

### 加班倍率规则
| 日期类型 | 倍率 | 说明 |
|----------|------|------|
| 普通工作日 | 1.5倍 | 周一至周五正常工作日 |
| 调休工作日 | 1.5倍 | 因节假日调整的工作日 |
| 周末 | 2.0倍 | 周六、周日 |
| 特殊调休日 | 2.0倍 | 特定的调休工作日 |
| 法定节假日 | 3.0倍 | 国家规定的法定假日 |
| 请假 | 1.5倍 | 按请假时长扣款 |

## 🐛 常见问题

**Q: 修改基础工资会影响历史记录吗？**  
A: 是的，系统会重新计算所有历史记录的金额，确保数据一致性。

**Q: 数据可以跨设备同步吗？**  
A: 目前不支持，数据存储在本地浏览器。请使用导出功能在设备间迁移数据。

**Q: 如何重置所有数据？**  
A: 使用"清除全部数据"按钮，或清除浏览器本地存储数据。

**Q: 支持哪些Excel版本？**  
A: 支持.xlsx格式，兼容Office 2007+及WPS等主流办公软件。
