# CPK 3.0 智能分析系统 - README

## 📊 项目简介

CPK 3.0 是一款专业的制程能力分析工具，专注于质量数据管理和CPK统计分析。它提供了数据上传、自动清洗、异常值剔除、CPK计算、可视化展示和报告导出等完整功能链，帮助质量工程师快速评估制程能力。

---

## ✨ 核心功能

### 1. 📁 数据上传与解析
- 支持 **CSV** 和 **XLSX** 格式文件上传
- 自动识别表头和数据列
- 智能解析规格列（支持正负数规格，如 `100±2`、`-100±20`）

### 2. 🔍 数据预处理
- **SN列过滤**：按指定长度过滤序列号
- **时间去重**：同一SN保留最新时间记录
- **自动去重统计**：显示原始数据去重行数

### 3. 🗑️ 智能异常值剔除
- 基于容差系数（可调）剔除超出规格范围的数据
- **按规格独立剔除**：每个规格独立判断，一个规格超标不影响其他规格
- **剔除数据查看器**：可查看、恢复或永久删除被剔除的数据
- 支持**批量恢复**和**批量永久删除**

### 4. 📊 CPK统计分析
自动计算每个规格列的完整统计指标：
- 最大值、最小值、平均值、标准差
- **CP（制程能力）**
- **CPK（制程能力指数）**
- 超标数量、超标率
- 剔除记录数

### 5. 🎨 可视化展示
- **CPK汇总报告**：CP/CPK值自动着色
  - 🟢 **深绿色**（≥ 1.67）：优秀
  - 🟡 **浅绿色**（1.30 ~ 1.67）：良好
  - 🔴 **红色**（< 1.30）：需改进
- **原始数据表**：超标单元格红色高亮
- **三种筛选模式**：
  - 全部：显示所有数据
  - 仅超标：只显示超标行
  - 剔除后：只显示正常数据

### 6. 📤 报告导出
- **导出剔除后数据 (XLSX)**：导出正常数据 + 统计信息
- **导出汇总报告 (XLSX)**：带背景色的专业报告
  - 工作表1：CPK汇总统计（含颜色评级）
  - 工作表2：超标明细
  - 工作表3：剔除数据列表
  - 工作表4：统计看板

### 7. 🔄 实时交互
- 参数调整（SN长度、容差系数）自动刷新
- 搜索框实时过滤数据
- 剔除数据恢复后界面即时更新

## 🚀 快速开始

### 1. 环境准备
- 现代浏览器（Chrome、Edge、Firefox 等）
- 无需安装，直接打开 HTML 文件即可运行

### 2. 使用步骤

#### 步骤一：上传数据
点击 **"📁 上传 CSV / XLSX"** 按钮，选择您的数据文件。

**数据格式要求：**
- 第一行为表头
- 包含 SN列 和 时间列
- 规格列需包含规格范围，格式如：`100±2`、`-100±20`

#### 步骤二：配置参数
- **SN列**：选择序列号列
- **时间列**：选择时间列（用于去重）
- **SN长度**：设置序列号长度（默认15）
- **容差系数**：调整剔除阈值（默认1.30）

#### 步骤三：查看分析结果
- **CPK汇总报告**：查看各规格的统计指标
- **原始数据表**：查看原始数据，超标行红色高亮
- **筛选模式**：切换"全部/仅超标/剔除后"

#### 步骤四：管理剔除数据
- 展开 **"🚫 剔除数据查看器"**
- 可单独恢复、批量恢复或永久删除剔除记录

#### 步骤五：导出报告
- **导出剔除后数据 (XLSX)**：导出正常数据
- **导出汇总报告 (XLSX)**：导出完整分析报告

## 📖 功能详解

### 规格列识别
系统自动识别包含 `±` 符号的列作为规格列，支持：
- 正数规格：`100±2`、`50±0.5`
- 负数规格：`-100±20`、`-200±10`

### 剔除逻辑
- **按规格独立剔除**：每个规格独立判断是否超出 [中心值 ± 容差系数 × 公差]
- **一行多个剔除记录**：一行数据可能因多个规格超标而有多个剔除记录
- **统计时区分**：计算CPK时，每个规格只排除自己对应的剔除记录

### CPK评级标准
| CPK值 | 评级 | 颜色 |
|---|---|---|
| ≥ 1.67 | 优秀 | 🟢 深绿色 |
| 1.30 ~ 1.67 | 良好 | 🟡 浅绿色 |
| < 1.30 | 需改进 | 🔴 红色 |

## ⚙️ 配置说明

### 容差系数调整
- **系数越小**：剔除越严格，更多数据被剔除
- **系数越大**：剔除越宽松，更少数据被剔除
- **建议值**：1.30（默认）

### SN长度设置
- 根据实际序列号长度设置
- 不符合长度的数据会被自动过滤

## 🐛 常见问题

### Q1: 上传文件后没有反应？
**A**: 检查文件格式是否为 CSV 或 XLSX，确保第一行为表头。

### Q2: 没有识别到规格列？
**A**: 确保规格列包含 `±` 符号，如 `100±2`。

### Q3: 剔除数据太多怎么办？
**A**: 调大"容差系数"值，减少剔除数量。

### Q4: 恢复的数据没有更新到汇总表？
**A**: 恢复后界面会自动刷新，如未更新请点击"刷新报告"。
