# 加班工时计算系统

## 概述

加班工时计算系统是一个基于Web的应用程序，专为记录和管理加班时间、请假信息而设计。它提供了直观的日历视图、详细的统计信息和Excel导入导出功能，帮助用户轻松跟踪工作时间和计算相关薪资。

## 🌟 主要功能

### 📅 智能日历视图
- **直观月历显示**：支持上月/下月快速切换
- **智能日期标记**：不同颜色区分节假日、周末、调休日和补班日
- **一键记录**：直接点击日期记录加班或请假信息
- **今日高亮**：当前日期特殊标识，便于快速定位

### 💰 精准薪资计算
- **灵活工资设置**：支持自定义标准工资（默认4900元）
- **自动薪资计算**：基于国家标准21.75个工作日计算日薪和时薪
- **智能倍率应用**：
  - 🟢 工作日/补班日加班：1.5倍（上限3小时/天）
  - 🔵 周末/调休日加班：2倍（上限11小时/天）
  - 🟣 节假日加班：3倍（上限11小时/天）
  - 🔴 请假扣款：1.5倍

### 📊 全面统计功能
- **多维度统计**：
  - 本周加班统计（周一到周日，上限20小时/周）
  - 本月加班统计（上限80小时/月）
  - 本季度加班统计（上限240小时/季）
  - 本年加班统计（上限960小时/年）
- **全局统计**：月/季/年/总计四维度进度条可视化
- **分类统计**：按加班类型（A/B/C）分别统计薪资
- **实发工资**：自动计算实发金额（标准工资 + 加班费 - 请假扣款）

### ⚠️ 智能加班限制提醒
- **每日限制**：工作日≤3h，周末/节假日≤11h
- **每周限制**：总计≤20小时
- **超限警告**：
  - 顶部红色横幅提醒
  - 日历格子红色高亮标记
  - 统计卡片脉冲动画
  - 进度条颜色变化（绿→橙→红）

### 🔄 强大数据管理
- **Excel导出**：生成专业Excel报表（加班明细表、统计汇总表）
- **数据导入**：支持从Excel文件恢复数据
- **批量操作**：多选模式一次性设置多个日期的加班/请假时长
- **双模式清除**：清除当月数据 / 全局清除所有数据
- **本地存储**：数据自动保存到localStorage，刷新不丢失

### 📱 移动端适配
- **响应式设计**：完美适配桌面和移动设备
- **触摸优化**：移动端输入框自动居中，遮罩层保护
- **日历精简**：移动端显示圆形指示器，节省空间

### 🌙 暗色模式
- **系统跟随**：自动适配系统深色/浅色模式
- **全局统一**：所有UI元素同步切换

### 📈 访问统计
- **不蒜子集成**：显示总访问量、独立访客数、今日访问次数
- **数据持久化**：访问记录保存在本地

### 🔧 自定义设置
- **加班事由**：自定义填写加班原因
- **填报人信息**：记录填报人姓名和工号
- **备案加班**：设置备案加班默认值

## 🚀 使用指南

### 基本操作
1. **设置基础工资**：在顶部输入框调整您的月薪
2. **记录考勤**：
   - 点击日历日期，输入正数记录加班
   - 输入负数记录请假
   - 输入0清除记录
3. **查看统计**：实时查看薪资计算和加班进度

### ⚡ 批量操作技巧
1. 点击"多选模式"进入批量模式
2. 在日历中点击选择多个日期（支持取消选择）
3. 输入统一时长，点击"应用设置"一键应用到所有选中日期
4. 支持加班和请假的批量设置

### 📤 数据备份与恢复
- **导出备份**：点击"导出Excel"生成包含明细和汇总的Excel文件
- **导入恢复**：点击"导入Excel"从备份文件恢复数据
- **跨月管理**：支持不同月份数据的独立管理

### 🗑️ 数据清除
- **清除当月数据**：只清除当前显示月份的数据
- **全局清除所有数据**：⚠️ 清除所有月份的所有记录（二次确认）

### 📊 查看全局统计
- 点击"全局加班统计"折叠卡片展开查看
- 支持本月/本季度/本年/总计四维度统计
- 进度条颜色直观展示使用率

## 📝 计算规则说明

### 薪资计算基础
- **月工作日**：21.75天（国家标准）
- **日薪计算**：月薪 ÷ 21.75
- **时薪计算**：日薪 ÷ 8小时

### 加班限制规则
| 日期类型 | 每日上限 | 每周上限 |
|----------|----------|----------|
| 工作日/补班日 | 3小时 | 20小时 |
| 周末/调休日 | 11小时 | 20小时 |
| 法定节假日 | 11小时 | 20小时 |

### 加班倍率规则
| 日期类型 | 倍率 | 说明 |
|----------|------|------|
| 普通工作日 | 1.5倍 | 周一至周五正常工作日 |
| 补班日 | 1.5倍 | 因节假日调整的工作日 |
| 周末 | 2.0倍 | 周六、周日 |
| 调休日 | 2.0倍 | 特定的调休工作日 |
| 法定节假日 | 3.0倍 | 国家规定的法定假日 |
| 请假 | 1.5倍 | 按请假时长扣款 |

### 全局统计限额
| 维度 | 限额 |
|------|------|
| 本月 | 80小时 |

## 🐛 常见问题

**Q: 修改基础工资会影响历史记录吗？**  
A: 是的，系统会重新计算所有历史记录的金额，确保数据一致性。

**Q: 数据可以跨设备同步吗？**  
A: 目前不支持，数据存储在本地浏览器。请使用"导出Excel"功能在设备间迁移数据。

**Q: 如何重置所有数据？**  
A: 使用"全局清除所有数据"按钮，或清除浏览器本地存储数据。

**Q: 加班超限了怎么办？**  
A: 系统会弹出警告，但您仍可选择强制保存。建议合理安排加班时间。

**Q: 周末加班上限为什么是11小时？**  
A: 正常上班8小时 + 加班3小时 = 11小时，符合劳动法规定。
