# 加班工时计算系统

一个专业、美观且功能完善的加班工时计算工具，支持日历视图、多选批量操作、Excel导入导出等功能。

## ✨ 主要功能

### 📅 日历管理
- **智能日历视图**：显示周末、节假日、调休日、补班日
- **多选批量操作**：支持批量设置加班/请假时长
- **直观显示**：不同加班类型使用不同颜色标识
- **今日高亮**：自动高亮显示今天日期

### 💰 工资计算
- **灵活工资设置**：支持自定义月标准工资
- **自动计算**：实时计算日薪、时薪
- **加班费率**：根据日期类型自动应用不同倍率
  - 工作日：1.5倍
  - 周末/调休日：2倍
  - 节假日：3倍

### 📊 数据管理
- **本地存储**：所有数据自动保存到浏览器本地存储
- **Excel支持**：
  - 导出详细加班记录和统计汇总
  - 导入Excel文件快速填充数据
- **批量操作**：支持多日期批量设置加班/请假
- **数据清除**：可按月份清除数据

### 📱 响应式设计
- **移动端优化**：完美适配手机和平板
- **触摸友好**：优化移动端操作体验
- **键盘支持**：支持快捷键操作

## 🚀 快速开始

### 基本使用步骤
1. **设置工资**：在左上角输入月标准工资
2. **添加记录**：
   - 单个日期：直接点击日期输入时长（正数加班，负数请假）
   - 批量设置：开启多选模式，选择多个日期后输入时长
3. **查看统计**：实时显示加班总时长、加班工资、请假扣款、净收入
4. **导出数据**：点击"导出Excel"下载完整报告

### 多选模式说明
1. 点击"多选模式"按钮进入多选状态
2. 点击日历日期进行选择/取消选择
3. 输入加班时长（正数加班，负数请假，0清除）
4. 点击"应用设置"完成批量操作

## 🎨 界面说明

### 颜色标识
- 🔵 **蓝色边框**：已设置加班/请假的日期
- 🟢 **绿色圆点**：加班记录
- 🔴 **红色圆点**：请假记录
- 🌟 **特殊标识**：
  - 红色"假"：节假日(3倍)
  - 橙色"调"：调休日(2倍)
  - 紫色"补"：补班日(1.5倍)

### 统计面板
- **总加班时长**：实际加班时长（扣除请假后）
- **加班工资**：分类显示A/B/C类加班工资
- **净收入**：标准工资 + 加班费 - 请假扣款
